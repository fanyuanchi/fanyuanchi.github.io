搭建自己的博客

​ —–15003范原驰

下载应用：

1.Node.js

2.Git

安装Hexo:

1.新建一个文件夹，我命名为“study_blog”。

2.与文件夹内右键打开Git Bash Here。

3.输入命令行：npm install -g hexo-cli ,等待hexo安装完成。

4.初始化文件夹：输入hexo init。

5.安装必备组件：输入npm install。

6.生成静态网页：输入hexo g。

7.打开本地服务器：输入hexo s,用浏览器输入https://localhost4000/查看。

注册Github:

1.注册Github。

2.点击new repository新建一个项目。

3.为项目命名，以“.github.io”为后缀。

连接Github和hexo：

1.右键打开Git Bash。

2.生成ssh key:输入ssh-keygen -t rsa -c”注册Github的邮箱”。

3.到C盘pub文件夹找到ssh码，或者去邮箱复制。

4.到刚刚建立的项目中，点击Settings，找到SSH and GPG keys。

5.将ssh码复制粘贴即可。

6.安装hexo-deployer-git:在Git Bash中输入npm install hexo-deployer-git。

7.将hexo部署到github pages:修改_config.yml文件末尾的代码：

​ deploy:

​ type: git

​ repository: git@github.com:用户名/用户名.github.io.git

​ branch: master

8.运行hexo:输入hexo d。

9.在浏览器中输入https://用户名.github.io/就可以看到hexo网站。

发布文章：

1.在博客文件根目录下右键打开Git Bash。

2.输入hexo n “article”。

3.用markdown编辑器在该文件中编写文章。

4.输入hexo g生成页面。

5.输入hexo d部署发布。

网址：https://fanyuanchi.github.io/
